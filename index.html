大会介绍

中国规模最大，最高水平，最有影响力的Ruby技术大会 - RubyConfChina 将在2012年11月17日至18日在上海杨浦区创智天地举行，本次大会是第四次Ruby中国技术大会。大会规模将超过500人，由 Ruby China 和 TopGeek 社区共同发起，并邀请了 Ruby 语言之父松本行弘先生，以及来来自海外，台湾，国内本土的最顶尖的 Ruby 开发高手，为国内所有关注 Ruby，使用 Ruby，推广 Ruby 的企业和个人，提供平等，开放的交流平台。本年度的 Ruby 大会是一场技术的饕餮盛宴，有超过20场的技术讲座，主题演讲，来自全国的Ruby 社区的组织者们将倾力为大家打造一场国内乃至亚洲顶级的Ruby大会。

讲师阵容

松本行弘 (Matz)

Ruby 2.0
Ruby 语言的发明人，1993 年发布了 Ruby 语言的第一个版本，随后 Ruby 逐渐在日本流行起来。2000年，Dave Thomas 出版了《Programming Ruby》一书，首次将这个发源于日本的动态编程语言介绍到了英文社区，Ruby 开始在全球范围内流行。04 年，丹麦人 DHH 用 Ruby 编写了 Web 快速开发框架 Ruby on Rails，彻底掀起了 Ruby 语言的热潮。

Fred Wu

如何通过努力和一些小聪明，成为一个更优秀的 Ruby 程序员
slim 模版语言开发者，为 Rails 等项目做过的贡献，一位对开源社区充满热情的程序员。出生和成长于上海，从13岁开始接触到并爱上了计算机与网络，从计算机杂志上“认识”了 HTML，萌发了对网络的憧憬。经过中学阶段对网络的痴迷到最终选择了互联网程序开发作为职业。毕业于澳大利亚墨尔本的 Monash 大学，就读于信息技术与系统（多媒体）专业。曾经在 Envato, PlayUp 和 SitePoint 等公司就职，目前在 Locomote 带领技术团队。

太檑 (Terry Tai)

How we test our projects
Pragmatic.ly, railscasts-china.com 创始人。典型的理想主义者。热爱开源，喜欢分享，热衷于把自己知道的东西以各种方式分享给他人。对代码有洁癖，且有点偏执。从小励志成为职业玩家的他最终走向了技术宅的不归之路，最可怕的是他还深深的热爱上了这份事业。

蔡望勤 (caiwangqing)

用 Rails / Sinatra 实现企业智能能源云服务
蔡望勤是 chinaonrails.com 的创始人。1997 年接触编程语言 Pascal，01 年至 05 年为珠三角多家知名企业研发知识管理系统和人力资源管理系统。05 年加入当时中国最大的社交网络平台 UUZone，从此专注互联网应用研发。07 年赴北京参与创建时尚社交网络公司 P1.cn，并担任 CTO。现在他住在深圳，任易皆能科技(深圳)有限公司云服务研发总监。
